# EchoDeck v0.1.2

EchoDeck v0.1.2 is a cross-platform cleanup release that normalizes the app name from the earlier macOS-focused package to EchoDeck.

## Highlights

- Normalized product naming to EchoDeck
- Confirms Windows and macOS as supported desktop build targets
- Maintains local-first music playback
- Includes Windows installer and portable EXE release assets
- Keeps playlists, queue, EQ, visualizers, themes, and source-aware streaming placeholders
- Prepares the project for the next v0.2.0 streaming-source MVP
