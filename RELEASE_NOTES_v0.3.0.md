# EchoDeck v0.3.0

EchoDeck v0.3.0 adds the first iOS Capacitor scaffold while preserving the existing desktop app.

## Highlights

- Adds Capacitor configuration
- Adds iOS setup, doctor, prepare, sync, and open scripts
- Adds initial app structure:
  - `apps/desktop`
  - `apps/ios`
  - `apps/web`
  - `packages/shared`
  - `packages/player-core`
  - `packages/themes`
  - `packages/visualizers`
- Adds iOS setup and Xcode documentation
- Adds iOS local file import plan
- Adds App Store / streaming compliance notes
- Keeps Windows/macOS desktop workflow intact
- Version bumped to 0.3.0

## Important

This is a scaffold release. Actual iOS build and testing must be done on macOS with Xcode.
