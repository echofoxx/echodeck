# EchoDeck v0.2.1 Visualizer Update

EchoDeck v0.2.1 adds a full-screen visualizer overlay and expands the visual modes available in the EQ / Visuals section.

## Modes

- Bars: classic spectrum analyzer
- Wave: waveform display
- VU: horizontal VU-style meters
- Radial: circular spectrum display
- Orbit: ambient orbital particle display

## Behavior

Local files use the Web Audio analyzer for audio-reactive visuals. Streaming embeds remain source-controlled, so the full-screen visualizer displays an ambient source-safe animation instead of inspecting protected streaming audio.

## Controls

- Now Playing: click the `⛶` button.
- EQ / Visuals: click **Full Screen**.
- Keyboard: press `F` to open and `Esc` to exit.
