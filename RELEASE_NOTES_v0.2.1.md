# EchoDeck v0.2.1

EchoDeck v0.2.1 is a layout and visualizer polish release.

## Highlights

- Adjusted the default desktop layout so Now Playing content fits cleanly in the window.
- Reduced oversized player/card minimums that caused text clipping on Windows desktop sizes.
- Added a full-screen visualizer overlay.
- Added Radial and Orbit visualization modes.
- Added full-screen visualizer controls from Now Playing and EQ / Visuals.
- Added ambient source-safe visuals for streaming entries.
- Updated default Electron window sizing to better match the refined layout.

## Visualizer modes

- Bars
- Wave
- VU
- Radial
- Orbit

## Windows downloads

- EchoDeck-0.2.1-Windows-Setup.exe
- EchoDeck-0.2.1-Windows-Portable.exe

## Notes

Local audio files drive the live audio-reactive visualizer. Streaming embeds remain source-controlled and are not inspected, modified, cached, or re-streamed.
