# EchoDeck v0.2.9

EchoDeck v0.2.9 is a desktop stabilization and release polish build. It is intended to be the final desktop-focused release before the v0.3.0 iOS Capacitor scaffold.

## Highlights

- Desktop release tools in Settings
- Duplicate scan for local metadata and streaming URLs
- Clear library confirmation workflow
- Recently played persistence panel
- Improved release documentation
- Screenshot structure for GitHub README polish
- iOS preparation docs and feature matrix
- Version bumped to 0.2.9

## Recommended validation

1. Enable Demo Mode.
2. Test all themes.
3. Test all deck modes.
4. Use Mobile Preview.
5. Scan for duplicates.
6. Build Windows installer and portable EXE.
7. Publish release assets to GitHub.

## Release assets

Expected Windows build output:

- `EchoDeck-0.2.9-Windows-Setup.exe`
- `EchoDeck-0.2.9-Windows-Portable.exe`
