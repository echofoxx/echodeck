# EchoDeck v0.2.7

EchoDeck v0.2.7 adds the new **Studio White** theme inspired by a minimal tape workstation / LCD music device aesthetic.

## Highlights

- New **Studio White** theme in Settings > Themes
- Soft white and silver hardware-inspired surfaces
- Monochrome LCD-inspired visual treatment
- Minimal industrial retro player styling
- Green status light and red accent slider styling
- Works with desktop and mobile preview layouts
- Preserves saved theme, deck mode, and playback preferences

## Included in this release

- Desktop source package update
- Windows packaging compatibility retained
- Existing Analog Cream, Modern Dark, Modern Light, Vintage, and Cassette themes remain available
