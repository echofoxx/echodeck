# EchoDeck v0.2.6

EchoDeck v0.2.6 focuses on mobile UX polish and playback/library persistence.

## Highlights

- Added Mobile Preview bottom tab navigation.
- Improved touch-sized mobile controls and compact card behavior.
- Added library source filter, favorites filter, and sorting by recent/title/artist/source/plays.
- Added live library result summary.
- Added queue save-as-playlist action.
- Added clear queue confirmation.
- Preserves last selected tab, queue, deck mode, mobile preview state, visualizer mode, volume, repeat, shuffle, source filter, and sort mode.
- Added iOS Capacitor planning documentation.
