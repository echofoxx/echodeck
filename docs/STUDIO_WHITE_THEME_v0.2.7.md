# EchoDeck v0.2.7 — Studio White Theme

Studio White is a minimal, hardware-inspired theme built around soft white chassis panels, silver edges, monochrome LCD styling, and restrained red/green signal accents.

## Visual characteristics
- Off-white surfaces
- Light gray / silver borders
- Monochrome text and iconography
- LCD-inspired display tones
- Red progress / transport accent
- Green status indicator accent

## Intended use
- Mobile Preview
- Minimal / lab / workstation deck layouts
- Compact player experiences
- Clean presentation screenshots
