# EchoDeck v0.2.8 — Theme Gallery

The Theme Gallery gives users a visual way to compare and apply EchoDeck themes.

## Features

- Preview cards for every built-in theme
- Recommended deck mode per theme
- One-click theme application
- Theme + deck presets
- Demo mode for screenshots and visual QA

## Built-in presets

- Analog Turntable
- Studio Mini LCD
- Dark Visualizer
- Vintage EQ
- Cassette Mixtape
