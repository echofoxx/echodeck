# EchoDeck v0.3.0 — App Store Notes

EchoDeck should avoid unsupported streaming behavior on iOS.

## Do not include

- YouTube audio extraction
- Stream ripping
- Ad bypass
- Protected media caching
- Hidden web players for prohibited use

## Safer path

- Local file playback
- User-selected files
- Official embeds/SDKs where allowed
- Apple Music path through official Apple APIs later
- Clear user privacy disclosures
