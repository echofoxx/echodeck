# EchoDeck iOS Streaming Limitations

Streaming platforms remain source-aware and compliance-bound on iOS.

## Do not implement

- Stream extraction
- YouTube-to-MP3 conversion
- Protected stream caching
- Ad bypass
- Hidden third-party players
- Re-hosting protected music

## Implement through official paths

- YouTube embeds / official APIs
- SoundCloud widgets/embeds
- Apple Music official integration path
- Spotify official integration path
