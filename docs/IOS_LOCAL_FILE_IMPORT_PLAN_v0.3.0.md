# EchoDeck v0.3.0 — iOS Local File Import Plan

Desktop EchoDeck can import files and folders more freely than iOS. iOS requires a mobile-safe file selection flow.

## Planned v0.3.1 behavior

- Use iOS document picker through Capacitor-compatible patterns
- Import selected audio files from the Files app
- Store metadata in local app storage
- Avoid unrestricted folder scanning
- Preserve user privacy by default

## Supported first targets

- MP3
- M4A/AAC where supported
- WAV where supported

## Later

- Better metadata extraction
- Artwork import
- iCloud Drive workflows
- Local/offline playlist sync
