# EchoDeck Roadmap

## v0.1.0 — Local Hi-Fi MVP

Status: built.

- Electron macOS source app
- Local files and folder import
- Library, queue, playlists
- EQ, visualizer, crossfade setting
- YouTube and SoundCloud URL embeds
- Themes and local backup

## v0.2.0 — Local Library Hardening

- SQLite local database
- Music folder watcher
- Album/artist/genre metadata extraction
- Artwork extraction
- Duplicate detection
- Smart playlists
- Recently played dashboard
- Mini-player window
- Global media key improvements

## v0.3.0 — YouTube Search + Better Streaming Cards

- YouTube Data API configuration
- In-app YouTube search results
- Better thumbnails/titles/durations
- Source cards and compliance labels
- Saved searches

## v0.4.0 — Spotify

- Spotify OAuth
- Spotify search
- Spotify playlist import
- Spotify Web Playback SDK player where supported
- Premium and policy guardrails

## v0.5.0 — Apple Music

- MusicKit developer token setup
- Apple Music authorization
- Catalog search
- Playlist and user library support

## v1.0.0 — Production Desktop Release

- Signed and notarized macOS app
- App icon and branding
- Installer DMG
- Full user guide
- Error monitoring option
- Privacy notice
- Automated build/release workflow

## v0.1.1 Windows Desktop Packaging

- Add Windows NSIS installer target.
- Add Windows portable EXE target.
- Add Windows build documentation and PowerShell helper scripts.
- Add GitHub Actions Windows build artifact generation.
