# EchoDeck v0.2.4 — Mobile Deck Modes + App Polish

EchoDeck v0.2.4 turns the Analog Cream direction into selectable player experiences and adds a mobile-preview workflow for the future iOS app.

## Highlights

- Added Deck Modes: Classic, Cassette, Turntable, Equalizer, Visualizer, and Mini Player.
- Added a Deck Mode strip on Now Playing for quick layout switching.
- Added Deck Layout settings card.
- Added Mobile Preview toggle to simulate a phone-sized iOS-style player shell.
- Added keyboard shortcuts: `D` cycles deck modes and `M` toggles Mobile Preview.
- Saved deck mode, mobile preview, last view, visualizer mode, theme, volume, EQ, queue, and playback preferences locally.
- Improved Windows/macOS app window minimum size so the layout can be tested in narrow/mobile proportions.
- Updated docs for mobile deck modes and app polish.
