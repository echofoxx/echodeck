# EchoDeck v0.2.4 Mobile Deck Modes

v0.2.4 adds selectable player layouts that are designed to support both desktop listening and the future iOS app path.

## Deck Modes

- **Classic** — balanced Now Playing layout.
- **Cassette** — Analog Cream cassette-inspired deck.
- **Turntable** — larger record/arm treatment.
- **Equalizer** — EQ-meter inspired stage.
- **Visualizer** — visualizer-forward layout.
- **Mini** — compact player for smaller windows.

## Mobile Preview

The Mobile Preview button constrains the main app area into a phone-like shape so the desktop app can be used to tune the future iOS interface before the Capacitor scaffold is created.

## Shortcuts

- `D` cycles deck modes.
- `M` toggles Mobile Preview.
- `F` opens the full-screen visualizer.
