# EchoDeck v0.2.5 — Mobile Preview Fit Fix

EchoDeck v0.2.5 fixes the mobile preview layout introduced in v0.2.4.

## Highlights

- Hides the desktop sidebar while Mobile Preview is active.
- Centers the app in a phone-sized canvas.
- Removes horizontal page bleed and unwanted body-level overflow.
- Makes the deck mode strip horizontally scrollable inside the phone canvas instead of forcing the whole window wider.
- Tightens mobile typography, controls, album stage, queue, and visualizer sizing.
- Improves Visualizer and Mini deck modes inside Mobile Preview.
- Keeps desktop layouts unchanged when Mobile Preview is off.

- EchoDeck-0.2.5-Windows-Setup.exe
- EchoDeck-0.2.5-Windows-Portable.exe
