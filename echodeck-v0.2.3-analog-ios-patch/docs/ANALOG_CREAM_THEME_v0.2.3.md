# EchoDeck Analog Cream Theme — v0.2.3

The Analog Cream theme is a warm, tactile interface direction for EchoDeck inspired by vintage cassette decks, turntables, EQ sliders, VU meters, and orange/cream hi-fi hardware.

## Design Goals

- Feel like a physical music device rather than a generic dashboard.
- Work well in compact desktop windows and future mobile layouts.
- Use warm cream surfaces, orange accents, soft shadows, and tactile cards.
- Keep controls readable and accessible.

## Theme Tokens

- Background: warm cream and tan gradients
- Primary: orange
- Secondary: amber
- Accent: dark vinyl black/brown
- Cards: translucent cream panels
- Controls: rounded, physical, soft-shadow buttons

## UI Patterns

- Vinyl record center stage for Now Playing
- Orange play/action controls
- Cream panels with subtle inner highlights
- EQ sliders and visualizer modes remain available
- Full-screen visualizer inherits theme colors
