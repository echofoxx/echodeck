# EchoDeck v0.2.8

EchoDeck v0.2.8 adds a Theme Gallery, one-click theme/deck presets, demo mode, and visual QA documentation.

## Highlights

- New Theme Gallery in Settings
- Theme preview cards for all built-in themes
- One-click Theme + Deck presets
- Demo Mode for screenshots and release validation
- Demo tracks, queue, and playlist seeding
- Screenshot folder structure for README and release media
- Visual QA checklist for theme/deck/mobile combinations

## Included themes

- Analog Cream
- Studio White
- Modern Dark
- Modern Light
- Vintage Receiver
- Cassette Deck

## Release assets

Build Windows assets with:

```powershell
npm run package:win
```

Expected outputs:

- `EchoDeck-0.2.8-Windows-Setup.exe`
- `EchoDeck-0.2.8-Windows-Portable.exe`
