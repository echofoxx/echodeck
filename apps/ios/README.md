# EchoDeck iOS App Scaffold

This folder marks the start of the EchoDeck iOS path.

The real native iOS project will be generated by Capacitor on macOS using:

```bash
npm run ios:prepare
npm run ios:add
npm run ios:sync
npm run ios:open
```

## Important

This scaffold does not mean the iOS app is fully functional yet. v0.3.0 prepares the repository, scripts, and documentation for the first Xcode-based build.
